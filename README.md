## 팩토리 메서드 패턴
### 구체적으로 어떤 인스턴스를 만들지 서브 클래스가 정한다.
- 다양한 구현체가 있고, 그중 특정한 구현체를 만들 수 있는 다양한 팩토리 제공

### 장점
- OCP 원칙 적용, 기존 코드를 건들지 않고 확장이 가능

### 단점
- 클래스가 늘어나는 단점
